# 🏥 Hospital Patient Management System

## 📌 Description
This project is a Hospital Patient Management System developed using Python. It simulates real-world hospital operations by managing patients, doctors, queues, and medical records efficiently.

The system uses multiple Data Structures concepts learned in class to ensure efficient data handling and processing.

---

## 🧠 Data Structures Used
- Linked List → Used to store and manage patient symptoms dynamically  
- Stack → Used to maintain patient medical history (LIFO order)  
- Queue → Used for handling routine checkup patients (FIFO)  
- Priority Queue → Used for emergency patients based on severity  
- Hash Map (Dictionary) → Used for fast lookup of patients and doctors  
- Arrays → Used to maintain department-wise statistics  

---

## ⚙️ Features
- Patient registration system  
- Add and track patient symptoms with severity  
- Emergency queue management (priority-based)  
- Routine checkup queue (FIFO system)  
- Doctor assignment and treatment handling  
- Medical history tracking for each patient  
- Full hospital report generation  
- Search patient by ID
